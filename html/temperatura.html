Temperatura


O que é temperatura?

A temperatura é uma forma de medir o quão frio ou quente está um corpo, ela é formada pela agitação das párticulas, e quanto mais rápido elas se movem maior a temperatura, ou quanto mais lentamente, mais frio será


O que é calor?

O calor é a energia que passa entre os corpos por conta da diferença de temperatura, não é algo que um corpo tem, mas que é transferido de um para o outro, além disso o calor sempre vai estar mais para quente do que para frio


Como medir?

A temperatura é medida com um sensor de temperatura, como os termômetros. Existem várias escalas, como Celsius (°C), que é utilizada no brasil, ou Fahrenheit (°F), que é utilizada nos EUA
